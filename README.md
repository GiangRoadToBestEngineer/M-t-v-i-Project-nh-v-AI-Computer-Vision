# Some Simple Project about AI, Computer Vision and Control system
Archive of projects built by Truong Giang and Gemini =))
# Features about project
* Identify and pinpoint objects with high probability.
* Displays drawings in the air on the screen.
* Inspect products for defects or passability using AI.
* Automating the plant growing process with liquid fertilizer.
# Tools and Libary
* Computer Vision: OpenCV , Numpy , YOLOv
* AI and Machine Learning: mediapipe , Tensorflow Keras Model
* GUI: tkinter
* System and Multithreading Support Team: winsound , threading
* Control Program PLC: GXworks2
